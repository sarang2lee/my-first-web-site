HTML iframe은 웹페이지 안에서 어떤 웹페이지를 보여주기 위해셔 사용합니다.

HTML iframe은 현재의 HTML 문서 안에 다른 문서를 품기 위해서 사용됩니다.

HTML iframe은 title 요소와 함께 사용하는 것이 좋은 습관입니다.

HTML iframe은 height and width 속성을 사용하며, pixel로 그 값이 규정됩니다, 이 속성을 사용하지 않으면 default값이 주어집니다.
