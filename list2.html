Chapter Summary

Use the HTML ul> element to define an unordered list. Use the CSS list-style-type property to define the list item marker. Use the HTML li> element to define a list item. Lists can be nested. List items can contain other HTML elements. Use the CSS property float:left to display a list horizontally
